# Changelog

All notable changes to PomoVault are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and versioning tracks [Semantic Versioning](https://semver.org/spec/v2.0.0.html) via `manifest.json`.

## [0.1.1] - 2026-05-02

### Changed

- Short plugin description updated for community directory guidelines (no redundant product name in the manifest tagline).

## [0.1.0] - 2026-05-02

### Added

- Single execution note (`PomoVault.md`) as the main workspace
- Built-in Pomodoro timer with work, short break, and long break; configurable durations and auto-advance
- Task list from a user-chosen markdown source, sorted by urgency and due date
- Priority badges (urgent through lowest) and one-click task completion with source file updates
- Plain markdown session logs in `PomoVault Log.md`
- Wikilink rendering in tasks; recurring-task metadata display
- Plugin settings for task source, timer lengths, sessions before long break, completion sound, and volume
- Mobile-capable (not desktop-only); no extra Obsidian plugins required
